cdef  keyword ==> 변수 선언, 함수 선언, 모듈 선언

--------------------------------------------------------
파이썬 변수는 메모리에 저장된 특정 객체를 가리키는 일종의 라벨

but cython에서는 오류!

--------------------------------------------------------
cython은 c에 존재하는 모든 자료형 지원

C structs, enums, typedefs  => 특수 자료형 정의
ps.  python object형으로 변수 선언시 어떤 객체 형태로도 변환가능

-------------------------------------------------------
cpdef  => cython 인터프리터 가능한 python 버전
             cython 에서 사용 가능한 c 버전
	 cpdef 사용법은 cdef 와 같다.

------------------------------------------------------
inline  => ex) cdef inline int max_inline(int a, int b):
		return a if a > b else b
 와 같이 사용하면 각 함수 호출문을 전체 함수 본문으로 대체할 수 있다.
